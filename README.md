# 💙 Maahyu
**AI-Powered Maternal Mental Wellness Platform**

*Maa (Mother) + Aayu (Life/Wellness)*

---

## Objective

Maahyu is a 24/7 AI-powered mobile app built to help Indian mothers detect, track, and manage postpartum depression — through clinical screening, daily mood tracking, and empathetic emotional support. It bridges the gap left by inaccessible, unaffordable, and culturally insensitive mental health care for mothers across India.

---

## Features

**MaaMind — Mental Wellness**
- Clinical screening using GAD-7 + PHQ-9
- Daily mood calendar with 6-emoji scale
- 7-day trend graph and 30-day pattern analysis
- Biweekly pregnancy progress tips by trimester
- Ajji's Wisdom flashcards — traditional Indian wellness + modern science
- Automated emergency alert to caregiver when score ≥ 20
- MaaWrapped — weekly personalised wellness insights

**MaaGang — Community**
- Anonymous peer community across 4 sub-groups
- Post, like, comment and reply
- Expert-moderated, auto-filtered for harmful content

**SHEro — AI Chatbot**
- 24/7 empathetic AI companion powered by Google Gemini
- Mood-aware responses using real-time MaaMind data
- 100+ pre-loaded pregnancy and postpartum FAQs
- Emergency keyword detection with instant crisis helpline routing
